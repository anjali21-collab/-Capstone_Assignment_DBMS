#  Car Rental Booking System(DBMS Project)

A full-stack web application that allows users to browse available cars, make bookings, and manage rentals with an admin-controlled dashboard.

---

## ✨ Features

### 👤 Customer Side
- View available cars with images and pricing
- Book cars by selecting date range
- View personal booking history
- Clean and responsive UI

### 👨‍💼 Admin Side
- View all bookings
- Mark bookings as Returned
- Cancel bookings
- Track booking status in real-time

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL (Local Database)

---

## ⚙️ Functionality

- Dynamic car availability system
- Booking status management (Booked / Returned / Cancelled)
- Real-time database updates
- Separate customer and admin workflows

---

## 📂 Project Structure

- `index.html` – Landing Page  
- `cars.html` – View Cars  
- `booking.html` – Book a Car  
- `myBookings.html` – Customer Bookings  
- `server.js` – Backend Server
